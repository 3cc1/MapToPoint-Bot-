# Event Map

Interactive map that displays events near a location with:
- Distance filtering
- Category search
- Date filtering
- Sidebar event list
- Click-to-zoom markers

## Tech Stack
- Leaflet.js
- JavaScript
- HTML/CSS
- Flask (optional backend)

## Features
- Location-based search
- Category-colored markers
- Sidebar interaction
